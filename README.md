# Hospital Management System - Flask Portfolio Project

A professional hospital management web application built using Python Flask and SQLite.

## Features
- Admin login system
- Dashboard statistics
- Doctor management
- Patient management
- Appointment scheduling
- Revenue reports
- Clean responsive UI

## How to Run
```bash
pip install -r requirements.txt
python app.py
```

Open in browser:
```text
http://127.0.0.1:5000
```
