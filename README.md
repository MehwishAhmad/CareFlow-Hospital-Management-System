
# CareFlow Hospital Management System

CareFlow is a modern Hospital Management System built with Python Flask and SQLite. It helps manage patients, doctors, appointments, reports, and daily hospital operations through a clean admin dashboard.

## Features

- Secure admin login
- Dashboard with analytics and charts
- Patient management
- Doctor management
- Appointment scheduling
- Appointment status update
- Search and filter functionality
- Revenue reports
- Clean and responsive UI
- SQLite database integration

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
- Chart.js

## Modules

### Dashboard
Shows total patients, doctors, appointments, revenue, charts, and recent appointments.

### Patients
Add, edit, delete, and search patient records.

### Doctors
Add, edit, delete, and search doctor records.

### Appointments
Create appointments, update appointment status, and manage appointment records.

### Reports
View daily revenue reports and appointment summaries.

## How to Run

1. Clone the repository

```bash
git clone YOUR_REPOSITORY_LINK
````

2. Install Flask

```bash
pip install flask
```

3. Run the project

```bash
python app.py
```

4. Open in browser

```bash
http://127.0.0.1:5000
```

## Project Purpose

This project was created as a portfolio-level Flask web application to demonstrate backend development, database integration, CRUD operations, dashboard design, authentication, search functionality, reporting, and real-world hospital workflow management.

The main purpose of CareFlow is to show how a hospital or clinic can manage patients, doctors, appointments, and reports through a simple, modern, and user-friendly web system.
